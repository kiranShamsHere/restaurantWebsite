## 📄 Description
A restaurant website designed using HTML, CSS, and JS, providing a visually appealing and user-friendly interface for customers to explore the restaurant's offerings.

## 🎉 Features
* Responsive design for optimal viewing on various devices
* Interactive homepage with scrolling effects
* Gallery section showcasing dishes and restaurant ambiance
* Contact page with form submission functionality
* Well-organized code structure for easy maintenance and updates

## 📚 Tech Stack
* HTML: For structuring content and creating the website's foundation
* CSS: For styling and layout, including responsive design and visual effects
* JS: For adding interactivity and dynamic functionality to the website

## 🗂️ Project Structure
The project consists of the following files and folders:
* `index.html`: The main entry point of the website
* `contact.html`: The contact page with form submission functionality
* `contact.css`: Stylesheet for the contact page
* `script.js`: JavaScript file for adding interactivity to the website
* Image files: Various images used throughout the website, including logos, dishes, and backgrounds

## 💻 Installation
1. Clone the repository using `git clone https://github.com/kiranShamsHere/restaurantWebsite.git`
2. Open the `index.html` file in a web browser to view the website

## 📊 Usage
* Open the website in a web browser to explore its features and functionality
* Use the navigation menu to access different pages and sections
* Fill out the contact form to submit inquiries or feedback
